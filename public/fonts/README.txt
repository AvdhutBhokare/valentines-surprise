Add Puccery Love font here
==========================

1. Download "Puccery Love" (e.g. Regular or Slapline) from:
   https://freefonts.co/fonts/puccery-love-slapline-regular

2. Save the .otf file in this folder as:
   PucceryLove.otf

3. Restart your dev server (npm start) if it's running.

The app will then use Puccery Love for all text.
